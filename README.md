# AutoCap Documentation Repository

This repository contains the documentation for AutoCap, an automatic DataCap allocator for the Filecoin network.

## Documentation

The full documentation is available in the [`docs/`](docs/) folder and is built with GitBook.

To view the documentation locally, see the GitBook structure starting at [`docs/README.md`](docs/README.md).

## About AutoCap

AutoCap enables Storage Providers to access Filecoin Plus (FIL+) DataCap through a transparent, automated, and activity-based process. DataCap is allocated based on verifiable on-chain economic activity.

Developed by [CryptoEconLab](https://cryptoeconlab.com/).

Visit [filautocap.xyz](https://filautocap.xyz/) to access the AutoCap Dashboard.
